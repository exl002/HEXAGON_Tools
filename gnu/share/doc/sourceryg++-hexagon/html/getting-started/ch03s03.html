3.3. Use with Dinkumware Libraries

Sourcery CodeBench Lite 5.1 for Hexagon ELF does not include C or C++ libraries or header files. This toolchain is intended to be used only with the Dinkumware libraries, which you must obtain and install separately.

Because the Dinkumware libraries and header files may be installed in an arbitrary location on your host computer system, you must explicitly specify their pathnames on your compile and link commands. For example, if you set the environment variable LIB to point at the root of your Dinkumware install tree, use: 

> hexagon-gcc options files \
  -I $LIB/include/common -I $LIB/include/common/c++ \
  -B $LIB/gnu/lib/gcc/hexagon/qdsp6/4.4.0 -B $LIB/dinkumware/lib
